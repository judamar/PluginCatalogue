# Smart Server Time

*如果本插件有用, 请给个star吧 :)*

## 特性

- 当**所有玩家**离开服务器后, *自动停止*服务器
- 大量节省您服务器的CPU :)
- 当有玩家**尝试连接**服务器时, *自动启动*服务器
- 支持**延时关闭**服务器
- 支持*手动启动/关闭*服务器
- 支持解析*Minecraft消息包*, 可以***正常返回**Ping请求*, 并对*尝试连接的玩家*给予服务器**状态提示**
